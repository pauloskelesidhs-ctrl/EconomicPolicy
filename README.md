# EconomicPolicy
I am recently working on a project that depicts three basic macroeconomic models of Economic Policy ( AD-AS , IS-LM , IS-MP) and the Solow Growth Model. The basic idea was born from my study on Greece’s long-run economic growth through the Solow model — particularly the path of recovery since the 2009 financial crisis. 



➡️The project is still in active development — new features and improvements are being added regularly.
Deployed on GitHub Pages with JavaScript.

It helps students and teachers explore:
- fiscal policy
- monetary policy
- demand shocks
- supply shocks
- stagflation
- equilibrium changes across macroeconomic models

## Features

- **AD-AS exercise diagram**
  - multiple AD and AS curves
  - numbered equilibrium points
  - dashed path arrows after each shock

- **IS-LM model**
  - old curves remain visible as dashed lines
  - new equilibrium shown clearly

- **IS-MP model**
  - demand-side analysis with monetary policy rule
  - equilibrium movement shown visually

- **Shock search engine**
  - type:
    - `positive demand shock`
    - `negative demand shock`
    - `positive supply shock`
    - `negative supply shock`
    - `stagflation`

## Why this project

This app is designed to make macroeconomic diagrams easier to understand visually.

Instead of only reading theory, users can:
- move curves
- compare old and new equilibria
- follow the path after a shock
- practice diagram-based economics exercises

## Models included

### AD-AS
Used for:
- inflation
- output
- supply shocks
- demand shocks
- stagflation

### IS-LM
Used for:
- fiscal policy
- monetary policy
- output and interest-rate changes

### IS-MP
Used for:
- modern demand-side analysis
- monetary-policy reaction
- output and policy-rate changes
